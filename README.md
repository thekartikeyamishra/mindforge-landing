## 📖 Overview

**MindForge: Battle of Minds** is a mobile competitive platform that bridges the gap between brain-training utilities and high-stakes competitive gaming. Built with a robust **Flutter** mobile client and a sleek **Next.js** web landing page, it features deterministic matchmaking, real-time multiplayer, and deep cognitive analytics. 

There are no dice rolls or pay-to-win mechanics. Every match is a pure test of intellect, backed by an authoritative server and strict anti-cheat firewall.

---

## ✨ Key Features

* ⚔️ **Ranked Duels & Multiplayer:** Real-time ELO-based matchmaking with phantom matching capabilities (Ghost injection) to ensure users never wait more than 3.5 seconds.
* 🎉 **Party Mode (Squad Chaos):** Custom lobbies to invite up to 5 friends using secure room codes.
* 🏆 **The Mirror Gauntlet (Tournaments):** High-stakes weekend events with zero RNG, identical constraint logic, and massive coin prize pools.
* 📊 **Weekly AI Cognitive Insights:** Advanced telemetry tracks reaction times and categorizes proficiencies (Logic, Math, Memory) to build a personalized cognitive profile.
* 🛡️ **Absolute Integrity (Anti-Cheat):** Sub-150ms reaction times are flagged. Protected by Firebase App Check (Play Integrity / DeviceCheck).
* 💎 **Mastery Auras & Store:** ELO-gated cosmetics and a fully integrated RevenueCat premium subscription model.
* 💰 **High-Yield Monetization:** Strategic Google AdMob integration (Smart Banners, Contextual Interstitials, and Rewarded Video "Intercepts").
